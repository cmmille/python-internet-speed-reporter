# Python Internet Complain Bot
Are you sick of complaining to ISPs about how terrible the internet is? Let Python handle those complaints for you!

# Instructions
* Unzip files
* In the same location as "main.py" create ".env" with the following variables:
    * username: Twitter username
    * password: Twitter password
* Run main.py 

Your internet speed will be assessed with [speedtest.net](https://www.speedtest.net/) and the result will be tweeted.
